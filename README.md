# ESP-FLY 项目

ESP-FLY 无人机相关资源汇总仓库。

## 目录说明

| 文件夹 | 说明 |
|--------|------|
| **2.3D** | 3D 模型文件，包含 ESP-FLY 无人机的外形模型（如 .f3z 格式） |
| **3.Firmware** | 固件工程，ESP-FLY 飞控固件（基于 Crazyflie 框架） |
| **5.Software** | 配套软件 |
| ↳ ESP-FLY-PC | 上位机调试软件，基于 PyQt6 开发，提供实时数据可视化、飞行控制、PID 调参等功能 |
| ↳ ESP-FLY-APP | Android 端控制 App，基于 Kotlin + Jetpack Compose 开发 |
